# Language Support for MacOS

#### Git

```bash
pip3 install gitlint
```

#### Bash

```bash
npm install --global bash-language-server
brew install \
    shfmt \
    shellcheck
```

#### Python

```bash
pip3 install \
    black \
    isort \
    docformatter \
    pyupgrade \
    ipython \
    pylint \
    flake8 \
    flake8-docstrings \
    darglint \
    mypy \
    rstcheck \
    rstfmt
```
